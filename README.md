# ATM管理系统

这是一个基于C++开发的ATM管理系统，具有用户注册、登录、存款、取款、转账等功能，同时支持管理员模式管理账户。

## 功能特性

- 用户注册与登录
- 存款、取款、转账
- 密码修改
- 账户挂失与解锁
- 交易记录查询
- 限额设置（单日限额、单笔限额）
- 管理员功能（查看所有账户、锁定/解锁账户、删除账户等）

## 快速开始

### 方法一：直接运行（无需编译环境）

直接运行 `atm_static.exe` 即可使用系统，此为静态编译版本，无需安装额外组件。

### 方法二：从源码编译

1. 确保已安装C++编译器（如MinGW-w64）
2. 打开终端，进入项目目录
3. 运行编译命令：

```bash
g++ -std=c++11 src/*.cpp -I include -o atm.exe
```

或者使用静态编译（推荐）：

```bash
g++ -std=c++11 src/*.cpp -I include -static -o atm.exe
```

## 管理员账户

- 账户：`666`
- 密码：`666`

## 注意事项

- 初始用户注册时会获得10000.00元余额
- 系统会自动保存账户数据到 `accounts.txt` 文件
- 管理员可以查看、锁定、解锁、删除账户等
- 用户密码错误3次将被锁定
- 系统支持每日限额和单笔限额设置
